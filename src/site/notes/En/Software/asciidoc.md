## backends 
###  slidy 
Nice HTML presentations with slidy / slidy2
* slidy2 gives some advantages http://csrp.iut-blagnac.fr/jmiwebsite/slidy2/slidy2_doc.html
* ditaa - [DIagrams Through Ascii Art](http://ditaa.sourceforge.net/#usage)
#### deckjs 
Another backend for nice HTML presentations with [deckjs](http://houqp.github.com/asciidoc-deckjs/) backend

 > [!info]
 > deckjs backend requires title attribute for images
* regular image
    * `image::foo.img[title="mytitle"]`
* ditaa requires
    * .My Title above '[ditaa]' section
